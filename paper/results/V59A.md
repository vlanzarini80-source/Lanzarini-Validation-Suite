# V59A — Interpretable Router Learning

## Purpose

This checkpoint searched for a simple interpretable backend-selection rule using the experimentally collected measurements.

The objective was to identify a compact decision rule capable of approximating the measured backend frontier.

---

## Preserved Results

| Metric | Value |
|--------|------:|
| Candidate rules evaluated | 41,762 |
| Best training accuracy | ~87.8% |

---

## Best Interpretable Rule

```text
Use FlashAttention when:

    T >= 13312

and

    W / T >= 0.045758929

Otherwise:

    use Lanzarini.
```

---

## Interpretation

The selected rule represented the best interpretable decision rule found among the evaluated candidates.

It approximated the measured backend frontier on the available training dataset.

---

## Limitations

This rule was fitted on the available measurements.

It should not be interpreted as:

- a universal routing rule;
- proof of generalization;
- a hardware-independent decision boundary.

Further validation is required on unseen configurations.
