# V57C — FlashAttention Correctness Validation

## Purpose

This checkpoint validates the numerical correctness of FlashAttention sliding-window attention against the SDPA local-mask reference implementation.

The objective is to verify that FlashAttention can be used as a trusted runtime baseline for subsequent performance comparisons.

---

## Hardware

- GPU: NVIDIA A100-SXM4-80GB
- Precision: FP16

---

## Reference Backend

Reference implementation:

- PyTorch SDPA local-mask

Compared backend:

- FlashAttention Sliding-Window

---

## Preserved Experimental Results

Validation status:

- Correctness: PASS

Observed numerical agreement satisfied the validation criteria adopted for the project.

No correctness failures were observed in the tested configurations.

---

## Interpretation

This checkpoint validates FlashAttention against the SDPA local-mask reference only for the experimentally evaluated configurations.

It establishes that FlashAttention was considered a numerically verified comparison baseline before subsequent runtime benchmarking.

---

## Limitations

This checkpoint does not establish:

- universal correctness;
- correctness outside the tested configurations;
- correctness for different GPUs;
- correctness for different precisions;
- correctness for arbitrary attention configurations.
