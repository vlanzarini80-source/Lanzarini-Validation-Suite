# Project Status

## Current Status

The public Lanzarini Validation Suite repository is actively maintained.

Version 1 is complete.

Version 2 is currently in progress.

The public repository currently provides:

- public validation framework;
- reproducibility pipeline;
- benchmark methodology;
- artifact integrity verification;
- scientific documentation;
- continuous integration through GitHub Actions.

The proprietary sparse-local attention kernel implementation is intentionally not included in this repository.
