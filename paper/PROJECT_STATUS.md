# Project Status

## Current Status

The Lanzarini Validation Suite is under active development.

The public repository currently contains:

- public validation framework;
- reproducibility pipeline;
- benchmark methodology;
- artifact integrity verification;
- documentation.

The proprietary sparse-local attention kernel is intentionally not included.
