# Security Policy

## Reporting a Security Issue

If you discover a security vulnerability related to the Lanzarini Validation Suite, please do not publish it publicly before contacting the maintainer.

Please open a private GitHub issue or contact the maintainer with:

- Description of the issue
- Steps to reproduce
- Affected version
- Possible impact
- Suggested mitigation (if available)

The maintainer will review the report and respond as soon as possible.

## Supported Versions

The latest version available on the main branch is considered the supported version for security updates.

## Scope

This repository contains public validation scripts, documentation and reproducibility material.

It does not contain proprietary kernels or confidential implementation details.
