# Completed Validation Reports

This directory contains completed Version 2 validation reports.

Each report follows the standard validation template and includes:

- experiment name;
- validation status;
- hardware;
- software environment;
- validation configuration;
- public artifacts;
- measured results;
- pass/fail criteria;
- known limitations.

Only experimentally verified reports should be placed in this directory.

Planned or in-progress work belongs outside this directory.
