# Public Benchmarks

## Purpose

This directory contains publicly distributable benchmark artifacts produced by the Lanzarini Validation Suite.

Only experimentally measured and publicly releasable results are included.

No simulated, estimated, or unpublished measurements are stored here.

---

## Repository Policy

Public benchmark artifacts may include:

- JSON benchmark reports
- CSV benchmark tables
- Runtime measurements
- Environment reports
- SHA-256 integrity manifests
- Validation reports

Only experimentally measured and publicly verifiable data are accepted.

---

## Excluded Data

This directory does not contain:

- proprietary Triton kernel source code;
- confidential implementation details;
- unpublished benchmark results;
- internal development experiments.

---

## Public Validation Principle

Every artifact stored in this directory should be:

- experimentally measured;
- reproducible;
- publicly distributable;
- documented.

---

## Future Releases

Future public releases may include additional benchmark artifacts as they become publicly available and experimentally validated.

No future benchmark is guaranteed until it has been experimentally verified and approved for public release.
