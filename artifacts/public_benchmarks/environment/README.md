# Public Benchmark Environment

## Purpose

This directory documents the execution environment associated with the public validation artifacts of the Lanzarini Validation Suite.

The canonical machine-readable environment report is available at:

- [`../../json/v1a_environment_report.json`](../../json/v1a_environment_report.json)

This page does not replace the JSON artifact. It provides a human-readable index to the canonical measured report.

---

## Verified Environment

The public V1A environment report records the following tested configuration:

| Component | Reported configuration |
|---|---|
| GPU | NVIDIA A100-SXM4-80GB |
| PyTorch | 2.4.1+cu124 |
| Triton | 3.0.0 |
| FlashAttention | 2.8.3.post1 |
| CUDA availability | Available |

These values are taken from the public V1A environment artifact and are not estimates.

---

## Scope

This environment report verifies the software and hardware configuration used for the corresponding public validation run.

It does not establish:

- performance portability to other GPUs;
- identical behavior on other software versions;
- universal runtime or energy results;
- validation of the proprietary Triton kernel in public-only mode.

## Integrity

The environment report is included in the public artifact workflow and may be referenced by SHA-256 integrity manifests generated by the Validation Suite.
